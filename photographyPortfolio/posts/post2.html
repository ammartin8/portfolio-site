Ways to Improve Sharpness of Images in Affinity Photo

December 10, 2019 by Amah

Recently, I have been learning how to improve my photo editing skills through Affinity Photo. Read to learn some great tips in making images look sharp and clear.

Unsharp Mask Method

  1. 1. Duplicate the background image
  2. 2. Go to Filters > Sharpen > Unsharp Mask
  3. 3. Increase Factor until image appears sharpen but not too sharpen, ~ 2.00ish. The factor is the amount of sharpening.
  4. 4. Then increase radius until image looks sharpen but not too exaggerated, make it appear natural.
  5. 5. Then bring lower the factor slider right at the point before it becomes blurry
  6. 6. The Threshold is usually kept at 0%. The threshold is used to adjust sharpness for pixels that considered high contrast when compared next to each other. The lower the contrast between the adjusted pixels the lower the threshold.


High Contrast Filter Method

  1. 1. Duplicate layer (Ctrl + J)
  2. 2. Filters > Sharpen > High Pass
  3. 3. Switch blend mode to linear light to see image.
  4. 4. Adjust radius until you have just enough sharpening.


Photo Edit Tip: You may apply sharpen again to get finer detail

  1. 1. Duplicate background layer and move to top layer
  2. 2. Filter > Sharpen > Highpass
  3. 3. Adjust radius slightly to get more fine detail.


Clarity Method

  1. 1. Duplicate background layer
  2. 2. Filter > Sharpen > Clarity
  3. 3. Adjust clarity until the image is just sharpened


Apply Clarity Method Non-destructively: Live filter Clarity

Remove duplicate layer and use only the background image. Then select the live filter layers and select clarity. Finally adjust in the same method as the first clarity method.



Tone Mapping Persona Method

  1. 1. Select Tone Mapping Persona at top left
  2. 2. Set tone compression and local contrast to 0.
  3. 3. Go to Detail Refinement in Tone map tab
  4. 4. increase radius slowly until sharpen
  5. 5. Set amount until natural and sharpen


Sharpen with Blur Method

  1. 1. Create two copies of background image
  2. 2. Select two top layers and group
  3. 3. Select group and select the blend mode to overlay
  4. 4. Select top layer in group and select ctrl + I to invert
  5. 5. Select that layer and blend mode and set to vivid light
  6. 6. With top merged layer select and go to live filter and select bilateral blur filter.
  7. 7. Slowly increase radius until just sharp
  8. 8. Slowly increase tolerance until sharp


Photo Editing Tip: Importing Macros

If you find your self using the same methods to edit your photos you can create your own macro or import some else's macro. Macros may be accessible to download online. Here are the basic steps to import a previously made macro:

  1. 1. Open Library in View > Studio > Libraries
  2. 2. Click top right corner and import macro

These methods were discovered on Scott Williams Photography Youtube Channel. To watch how he demonstrates each of these methods, please check him out at Scott Williams Photography! Thanks for reading!
